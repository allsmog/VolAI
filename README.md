# VolAI

**AI-Powered Memory Forensics Companion for Volatility3**

Auto-analysis + interactive chat-based investigation of memory dumps. Pluggable LLM backend (Claude, OpenAI, local models).

## Features (Planned)
- Automated memory dump triage and analysis
- Interactive chat-based forensic investigation
- Pluggable LLM backends (Claude, OpenAI, local)
- Built on top of Volatility3
- Structured reporting and artifact extraction
